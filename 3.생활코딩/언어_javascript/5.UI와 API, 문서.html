생활 코딩 강의 링크

자신이 필요한 것을 찾아내는 능력이 필요하다.

  • 1.래퍼런스와 튜토리얼(안내서)

    사전에 알고 있어야 하는 것을 튜토리얼을 통해 충분히 공부해야한다. 그 지식을 기반으로 해서 환경에 제공하는 명령들을 찾아야 한다.
    튜토리얼 - 문법
    래퍼런스 - 영어 사전, 국어 사전 같은 느낌

  • 2. 래퍼런스를 보는 방법

    자바스크립트의 래퍼런스는 2가지가 있다.
    2-1. 자바스크립트 자체 API
    날짜를 알아낼 수 있는 API, MATH, 정규표현식 등등이 있다.

    2-2. 자바스크립트가 동작하는 호스트 환경 API (web, node.js, google Apps scipt)
